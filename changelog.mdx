<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Launched "GQ Fitness Challenge" interactive series with progress trackers and workout videos.
- Integrated social sharing for article highlights directly from reading view.

## Breaking Changes

- Updated article metadata schema; migrate your integrations to the new format.

<CodeGroup tabs="JavaScript,Python">
```javascript
// Old format
const article = { id: '123', title: 'Style Guide' };

// New format (v2.0+)
const article = {
  id: '123',
  title: 'Style Guide',
  series: 'Style Essentials',
  publishedAt: '2024-10-01T10:00:00Z'
};
```
```python
# Old format
article = {"id": "123", "title": "Style Guide"}

# New format (v2.0+)
article = {
    "id": "123",
    "title": "Style Guide",
    "series": "Style Essentials",
    "publishedAt": "2024-10-01T10:00:00Z"
}
```
</CodeGroup>
