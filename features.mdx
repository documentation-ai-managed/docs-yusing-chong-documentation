Integrate GQ content into your apps with simple embeds.

<CodeGroup tabs="JavaScript,HTML">
  ```javascript
  const embedGQArticle = (articleId) => {
    const iframe = document.createElement('iframe');
    iframe.src = `https://www.gq.com/embed/article/${articleId}`;
    iframe.width = '100%';
    iframe.height = '600';
    document.body.appendChild(iframe);
  };
  embedGQArticle('mens-style-guide-2024');
  ```
  ```html
  <iframe
    src="https://www.gq.com/embed/article/mens-style-guide-2024"
    width="100%"
    height="600"
    frameborder="0">
  </iframe>
  ```
</CodeGroup>
